# PerconaLive 2014 - MariaDB multi-source replication

----

## Requirements

The requirements for running this vagrantfile are:

- VirtualBox 4.3.18 for guest additions
- Vagrant 1.5+

----

## Installation

```
  git clone git@github.com:dtest/pluk14-mariadb.git
  cd pluk14-mariadb
  vagrant up
```
